# Task 1: System Prompt Architect – Luxury Travel Consultant AI

## Project Overview

This project demonstrates the practical application of Prompt Engineering techniques to transform a general-purpose Large Language Model (LLM) into a specialized Luxury Travel Consultant AI. The system was designed using Persona Engineering, System Prompting, Constraint-Based Prompting, and Few-Shot Prompting techniques.

The AI assistant, named **Aurora Elite Travel Concierge**, provides personalized luxury travel recommendations while maintaining a professional and premium brand image.

---

## Problem Statement

General-purpose AI models often lack domain-specific expertise and consistent behavior. In the luxury travel industry, maintaining professionalism, customer satisfaction, and brand identity is critical. Existing AI systems may generate inconsistent responses, discuss competitors, or fail to follow business policies.

This project addresses these challenges by designing a robust system prompt that enforces a predefined persona, communication style, behavioral constraints, and security guardrails.

---

## Objectives

* Design a professional Luxury Travel Consultant AI.
* Implement Persona Engineering techniques.
* Apply System Prompting and Few-Shot Prompting.
* Establish behavioral constraints and knowledge boundaries.
* Prevent prompt injection and jailbreak attempts.
* Maintain a consistent luxury brand voice.
